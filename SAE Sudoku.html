SAE Sudoku

Objectif Principal

Créer une application Java permettant de jouer à un jeu de Sudoku contre une IA. Ce projet permet à l'utilisateur de jouer en mode solo ou compétitif contre une IA stratégique.

Fonctionnalités du Projet

  • Génération de Grilles de Sudoku: Création de grilles avec différents niveaux de difficulté.
  • Mode de Jeu Solo: Résolution libre avec validation des coups et possibilité d'indices.
  • Mode Compétitif: Jouer contre l'IA tour à tour.
  • IA de Résolution: Utilisation d’algorithmes comme le backtracking pour résoudre le Sudoku.
  • Interface Utilisateur: Interface intuitive avec JavaFX ou Swing.

Architecture Technique

  • Backend: Java
  • Frontend: Affichage dans le terminal
