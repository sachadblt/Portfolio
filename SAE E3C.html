SAE E3C

Objectif Principal

Développer une application Java permettant de jouer au jeu "Set!" en mode solo ou avec une IA, avec des fonctionnalités telles que la génération automatique de cartes et la détection de "sets" valides.

Fonctionnalités du Projet

  • Génération du Deck de Cartes: Création des 81 cartes avec extensions et combinaisons d'attributs (forme, couleur, nombre, remplissage).
  • Mode Solo: Résolution du jeu avec un système de points.
  • Mode IA: Résolution automatique du jeu par l’ordinateur.
  • Détection des Sets: Algorithmes pour détecter les "sets" valides.
  • Système de Scores: Attribution de points et gestion des pénalités pour mauvaises réponses.

Architecture Technique

  • Backend: Java
  • Frontend: Affichage dans le terminal
