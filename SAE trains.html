SAE Trains

Objectif Principal

Développer une application Java pour jouer à une version numérique du jeu de société Trains. Ce jeu permet à 2 à 4 joueurs de s’affronter en mode multijoueur local sur un même appareil.

Description du Jeu

Trains est un jeu de deck-building et de plateau où les joueurs développent des réseaux ferroviaires, construisent des gares et gèrent leurs cartes pour maximiser leurs points. Il mêle stratégie et gestion sur un plateau interactif.

Fonctionnalités de l’Application

  • Multijoueur Local: Mode de 2 à 4 joueurs jouable sur un même appareil.
  • Plateau de Jeu Virtuel: Représentation fidèle du plateau physique.
  • Gestion de Deck: Achat, tri et gestion des cartes "déchets".
  • Interface Intuitive: Design ergonomique pour manipuler les cartes et interagir avec le plateau.
  • Statistiques: Suivi des scores de chaque joueur.

Architecture Technique

  • Langage: Java
  • Interface Graphique: JavaFX
